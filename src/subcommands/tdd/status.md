## 詳細表示（-v オプション）

### 追加情報
- ストーリー別進捗（受け入れ基準の達成状況）
- 本日の統計（作業時間、サイクル数、平均時間）
- 品質指標（Fake It 使用率、Tidy First 遵守率）
- フィードバック履歴
- 現在作業中の詳細

### データソース
- イテレーションファイル: 進捗状況
- セッション JSON: 統計情報
- Git ログ: コミット履歴
- ストーリーファイル: 受け入れ基準

## 健康状態インジケーター

🟢 健全（必須ゲート > 90%）
🟡 注意（70-90%）
🔴 要改善（< 70%）

- ファイルが見つからない: 初期化を促す
